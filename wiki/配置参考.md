# 配置参考

---

## 配置文件层级

```
application.yml              # 基础配置
  ├── application-dev.yml     # 开发环境
  ├── application-test.yml    # 测试环境
  ├── application-prod.yml    # 生产环境
  └── bootstrap.yml           # Nacos 配置（优先级最高）

Nacos 配置中心
  ├── application-common.yml     # 公共配置
  ├── application-datasource.yml # 数据源配置
  └── application-redis.yml      # Redis 配置
```

---

## 核心配置项

### Spring Boot 基础

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `server.port` | 各服务不同 | 服务端口 |
| `spring.application.name` | - | 服务名（Nacos 注册名） |
| `spring.profiles.active` | `dev` | 激活的配置文件 |
| `spring.main.allow-bean-definition-overriding` | `true` | 允许 Bean 覆盖 |

### 数据源配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `spring.datasource.url` | - | JDBC URL |
| `spring.datasource.username` | - | 数据库用户名 |
| `spring.datasource.password` | - | 数据库密码 |
| `spring.datasource.driver-class-name` | `com.mysql.cj.jdbc.Driver` | 驱动 |

### Druid 连接池

| 配置项 | 推荐值 | 说明 |
|--------|--------|------|
| `spring.datasource.druid.initial-size` | 5 | 初始连接数 |
| `spring.datasource.druid.min-idle` | 5 | 最小空闲连接 |
| `spring.datasource.druid.max-active` | 20 | 最大活跃连接 |
| `spring.datasource.druid.max-wait` | 60000 | 获取连接超时（ms） |
| `spring.datasource.druid.time-between-eviction-runs-millis` | 60000 | 检测间隔 |
| `spring.datasource.druid.min-evictable-idle-time-millis` | 300000 | 最小空闲时间 |

### MyBatis Plus

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `mybatis-plus.configuration.map-underscore-to-camel-case` | `true` | 下划线转驼峰 |
| `mybatis-plus.configuration.cache-enabled` | `false` | 二级缓存 |
| `mybatis-plus.global-config.db-config.id-type` | `ASSIGN_ID` | 主键策略 |
| `mybatis-plus.global-config.db-config.logic-delete-field` | `deleted` | 逻辑删除字段 |
| `mybatis-plus.mapper-locations` | `classpath*:mapper/**/*.xml` | Mapper XML 路径 |

### Flyway

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `spring.flyway.enabled` | `true` | 启用 Flyway |
| `spring.flyway.baseline-on-migrate` | `true` | 基线迁移 |
| `spring.flyway.locations` | `classpath:db/migration` | 脚本路径 |
| `spring.flyway.table` | `flyway_schema_history` | 历史表名 |
| `spring.flyway.clean-disabled` | `true` | 禁用 clean |

### Redis

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `spring.data.redis.host` | `localhost` | Redis 地址 |

| `spring.data.redis.database` | `0` | 数据库编号 |

## Nacos 共享配置

项目在 `config/nacos/` 目录下提供了 Nacos 配置模板：

| 文件 | 说明 |
|------|------|
| `application-common.yml` | 公共配置 |
| `application-datasource.yml` | 数据源配置 |
| `application-redis.yml` | Redis 配置 |

将这些配置导入 Nacos 控制台后，所有微服务可共享同一份配置。
