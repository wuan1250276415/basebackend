## `@DistributedLock` 分布式锁

**所属模块**：`common-lock`

**功能**：对方法加分布式锁，防止并发执行

```java
@DistributedLock(
    key = "'user:update:' + #userId",   // SpEL 表达式
    waitTime = 5,                        // 等待获取锁超时（秒）
    leaseTime = 30,                      // 持有锁最长时间（秒）
    timeUnit = TimeUnit.SECONDS
)
public void updateUser(Long userId, UserUpdateDTO dto) {
    // 同一 userId 不会并发执行
}
```

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `key` | String | 必填 | 锁 Key，支持 SpEL |
| `waitTime` | long | 3 | 等待获取锁超时 |
| `leaseTime` | long | 30 | 持有锁最长时间 |
| `timeUnit` | TimeUnit | SECONDS | 时间单位 |

## `@Idempotent` 幂等性

**所属模块**：`common-idempotent`

**功能**：防止接口重复提交

```java
// TOKEN 策略
@Idempotent(strategy = IdempotentStrategy.TOKEN)
@PostMapping("/orders")
public Result<OrderVO> createOrder(@RequestBody OrderDTO dto) { ... }

// PARAM 策略
@Idempotent(strategy = IdempotentStrategy.PARAM, timeout = 5, timeUnit = TimeUnit.MINUTES)
@PostMapping("/payment")
public Result<Void> pay(@RequestBody PayDTO dto) { ... }

// SPEL 策略
@Idempotent(strategy = IdempotentStrategy.SPEL, key = "'pay:' + #dto.orderId")
@PostMapping("/confirm")
public Result<Void> confirm(@RequestBody ConfirmDTO dto) { ... }
```

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `strategy` | IdempotentStrategy | TOKEN | 策略：TOKEN / PARAM / SPEL |
| `key` | String | "" | SpEL 表达式（SPEL 策略必填） |
| `timeout` | long | 5 | 幂等 Key 有效期 |
| `timeUnit` | TimeUnit | MINUTES | 时间单位 |
| `message` | String | "请勿重复提交" | 重复请求提示消息 |

## `@DataScope` 数据权限

**所属模块**：`common-datascope`

**功能**：自动在 SQL 中追加数据权限条件

```java
// 本部门数据
@DataScope(type = DataScopeType.DEPT)
public List<UserVO> listUsers() { ... }

// 本部门及下级
@DataScope(type = DataScopeType.DEPT_AND_BELOW)
public List<OrderVO> listOrders() { ... }

// 仅本人数据
@DataScope(type = DataScopeType.SELF)
public List<TaskVO> listMyTasks() { ... }

// 自定义（需配合角色配置）
@DataScope(type = DataScopeType.CUSTOM, deptAlias = "u", userAlias = "u")
public List<UserVO> selectUserList(UserQuery query) { ... }
```

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `type` | DataScopeType | ALL | ALL / DEPT / DEPT_AND_BELOW / SELF / CUSTOM |
| `deptAlias` | String | "" | 部门表别名 |
| `userAlias` | String | "" | 用户表别名 |

## `@RateLimit` 限流

**所属模块**：`common-ratelimit`

**功能**：接口访问频率限制

```java
// 滑动窗口：每分钟最多 100 次
@RateLimit(key = "'api:users'", count = 100, period = 60, algorithm = RateLimitAlgorithm.SLIDING_WINDOW)
@GetMapping("/users")
public Result<List<UserVO>> list() { ... }

// 令牌桶：每秒 10 个令牌
@RateLimit(key = "'api:orders'", count = 10, period = 1, algorithm = RateLimitAlgorithm.TOKEN_BUCKET)
@PostMapping("/orders")
public Result<OrderVO> create(@RequestBody OrderDTO dto) { ... }

// 基于用户的限流
@RateLimit(key = "'upload:' + #userId", count = 5, period = 60, message = "上传过于频繁")
public Result<Void> upload(Long userId, MultipartFile file) { ... }
```

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `key` | String | 必填 | 限流 Key，支持 SpEL |
| `count` | int | 100 | 时间窗口内允许的请求次数 |
| `period` | int | 60 | 时间窗口（秒） |
| `algorithm` | RateLimitAlgorithm | SLIDING_WINDOW | SLIDING_WINDOW / TOKEN_BUCKET / FIXED_WINDOW |
| `message` | String | "访问过于频繁" | 限流提示消息 |

## `@RequiresPermission` 权限校验

**所属模块**：`basebackend-security`

**功能**：校验当前用户是否拥有指定权限

```java
// 单个权限
@RequiresPermission("user:list")
@GetMapping("/users")
public Result<List<UserVO>> list() { ... }

// 多权限 AND
@RequiresPermission(value = {"user:edit", "user:assign"}, logical = Logical.AND)
@PutMapping("/users/{id}/role")
public Result<Void> assignRole(...) { ... }

// 多权限 OR
@RequiresPermission(value = {"user:delete", "admin"}, logical = Logical.OR)
@DeleteMapping("/users/{id}")
public Result<Void> delete(...) { ... }
```

---

## `@RequiresRole` 角色校验

**所属模块**：`basebackend-security`

**功能**：校验当前用户是否拥有指定角色

```java
@RequiresRole("admin")
@GetMapping("/system/config")
public Result<Config> getConfig() { ... }

@RequiresRole(value = {"admin", "manager"}, logical = Logical.OR)
@PostMapping("/system/config")
public Result<Void> updateConfig(...) { ... }
```

## `@Sensitive` 敏感数据加密

**所属模块**：`basebackend-database`

**功能**：字段级别的加密存储与脱敏展示

```java
@Data
@TableName("sys_user")
public class User extends BaseEntity {

    @Sensitive(type = SensitiveType.PHONE)     // 138****8888
    private String phone;

    @Sensitive(type = SensitiveType.ID_CARD)   // 110***********1234
    private String idCard;

    @Sensitive(type = SensitiveType.EMAIL)     // a***@example.com
    private String email;
}
```

## `@DS` 动态数据源

**所属模块**：`basebackend-database`

**功能**：切换数据源

```java
@DS("master")   // 主库
public void write() { ... }

@DS("slave")    // 从库
public Object read() { ... }
```

---

## `@MasterOnly` 强制主库

**所属模块**：`basebackend-database`

**功能**：强制从主库读取（需实时一致性场景）

```java
@MasterOnly
public User getLatestUser(Long id) {
    return userMapper.selectById(id);
}
```

## Spring 常用注解

| 注解 | 说明 | 示例 |
|------|------|------|
| `@Transactional` | 事务管理 | `@Transactional(rollbackFor = Exception.class)` |
| `@Cacheable` | 查询缓存 | `@Cacheable(value = "user", key = "#id")` |
| `@CachePut` | 更新缓存 | `@CachePut(value = "user", key = "#dto.id")` |
| `@CacheEvict` | 清除缓存 | `@CacheEvict(value = "user", key = "#id")` |
| `@Async` | 异步执行 | `@Async("taskExecutor")` |
| `@Scheduled` | 定时任务 | `@Scheduled(cron = "0 0 2 * * ?")` |
| `@Valid` / `@Validated` | 参数校验 | 用于 Controller 参数 |
| `@GlobalTransactional` | Seata 分布式事务 | 跨服务事务 |
