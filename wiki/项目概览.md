# 项目概览

> BaseBackend 是一套基于 Java 17 + Spring Boot 3.1.5 + Spring Cloud 2022.0.4 的企业级微服务基础后台架构，采用 RBAC 权限模型，集成工作流引擎、消息队列、分布式事务、可观测性、文件存储等企业核心能力。

---

## 核心特性

### 微服务架构
- 基于 Spring Cloud Gateway 统一入口，Nacos 注册/配置中心
- 服务间通过 Feign 进行 RPC 调用
- Seata AT 模式分布式事务保障数据一致性

### 安全体系
- JWT 双 Token 机制（Access Token + Refresh Token）
- RBAC 权限模型，支持动态权限加载
- 数据权限（`@DataScope`），支持 ALL / DEPT / DEPT_AND_BELOW / SELF / CUSTOM 五种范围
- XSS / CSRF / SQL 注入多层防护
- 密钥轮换、Token 黑名单、多设备管理

### 数据层
- MyBatis Plus 3.5.5 ORM，支持逻辑删除、自动填充、乐观锁
- Flyway 数据库版本迁移
- 多级缓存（L1 本地 Caffeine + L2 Redis）
- 分布式锁（`@DistributedLock`）、限流（`@RateLimit`）、幂等性（`@Idempotent`）

### 消息与事件
- RocketMQ 5.2.0 消息队列集成
- 事务消息、幂等消费、死信队列
- 领域事件可靠发布（EventStore + 重试 + 指数退避）

### 可观测性
- OpenTelemetry + Micrometer 指标采集
- Prometheus + Grafana 监控仪表盘
- Loki 日志聚合
- Tempo 分布式链路追踪

### 工作流
- Camunda BPM 7.21.0 工作流引擎
- 流程定义、任务管理、历史查询

### 开发工具
- 代码生成器（FreeMarker / Velocity 模板）
- 数据库备份（MySQL / PostgreSQL，增量备份，S3 存储）
- Knife4j / SpringDoc OpenAPI 接口文档

## 适用场景

- **企业后台管理系统**：用户管理、角色权限、部门组织、系统配置
- **SaaS 多租户平台**：内置多租户支持，数据权限隔离
- **微服务中台**：标准化的基础设施库，可快速衍生业务微服务
- **工作流审批系统**：Camunda 工作流引擎，支持复杂业务流程
- **技术学习参考**：涵盖企业级微服务开发的主流技术栈实践
