# 文件服务

---

## 概述

`basebackend-file-service` 提供文件上传、下载、管理功能，支持多种存储后端。文件存储 SPI 定义在 `common-storage` 模块中，支持 Local、MinIO、S3、OSS 四种 Provider。

---

## 架构

```
客户端
  │
  ▼
Gateway ──→ File Service ──→ StorageService (SPI)
                                    │
                    ┌───────────────┼───────────────┐
                    │               │               │
              LocalStorage    MinIOStorage     S3Storage
              (本地文件)      (MinIO对象存储)  (AWS S3)
                                                    │
                                              OSSStorage
                                              (阿里云OSS)
```

---

## 多存储 Provider

### Provider 对比

| Provider | 适用场景 | 依赖 | 配置项 |
|----------|---------|------|--------|
| `Local` | 开发/测试环境 | 无 | `storage.local.path` |
| `MinIO` | 自建对象存储 | MinIO SDK 8.5.7 | `endpoint`, `access-key`, `secret-key`, `bucket` |
| `S3` | AWS 云环境 | AWS SDK | `region`, `access-key`, `secret-key`, `bucket` |
| `OSS` | 阿里云环境 | OSS SDK | `endpoint`, `access-key`, `secret-key`, `bucket` |

### 配置示例

```yaml
# MinIO 配置
storage:
  type: minio
  minio:
    endpoint: http://localhost:9000
    access-key: ${MINIO_ACCESS_KEY:admin}
    secret-key: ${MINIO_SECRET_KEY:admin123456}
    bucket: basebackend
    region: us-east-1

# 本地存储配置
storage:
  type: local
  local:
    path: /data/files
    url-prefix: http://localhost:8082/files
```

## 文件上传

### API 接口

```
POST /api/files/upload
Content-Type: multipart/form-data

参数：
  file: 文件（必须）
  directory: 存储目录（可选，默认 "default"）
```

### 使用示例

```java
@RestController
@RequestMapping("/api/files")
public class FileController {

    @Autowired
    private StorageService storageService;

    @PostMapping("/upload")
    public Result<FileVO> upload(@RequestParam("file") MultipartFile file,
                                 @RequestParam(value = "directory", defaultValue = "default") String directory) {
        // 生成唯一文件名
        String fileName = UUID.randomUUID() + getExtension(file.getOriginalFilename());

        // 上传到存储
        String fileUrl = storageService.upload(file.getInputStream(), directory + "/" + fileName);

        // 保存文件元数据
        FileInfo fileInfo = fileService.saveMetadata(file, fileUrl);

        return Result.success(toVO(fileInfo));
    }
}
```

## 文件下载

```java
@GetMapping("/download/{fileId}")
public void download(@PathVariable Long fileId, HttpServletResponse response) {
    FileInfo fileInfo = fileService.getById(fileId);

    response.setContentType(fileInfo.getContentType());
    response.setHeader("Content-Disposition",
        "attachment; filename=" + URLEncoder.encode(fileInfo.getOriginalName(), StandardCharsets.UTF_8));

    try (InputStream is = storageService.download(fileInfo.getStoragePath())) {
        IOUtils.copy(is, response.getOutputStream());
    }
}
```

## 文件管理

| 操作 | API | 说明 |
|------|-----|------|
| 上传 | `POST /api/files/upload` | 上传文件 |
| 下载 | `GET /api/files/download/{id}` | 下载文件 |
| 删除 | `DELETE /api/files/{id}` | 删除文件 |
| 查询 | `GET /api/files/{id}` | 查询文件信息 |
| 列表 | `GET /api/files` | 文件列表（分页） |

## MinIO 管理

MinIO 控制台地址：`http://localhost:9001`

```bash
# Docker 启动 MinIO
docker run -d --name minio \
  -p 9000:9000 \
  -p 9001:9001 \
  -e MINIO_ROOT_USER=admin \
  -e MINIO_ROOT_PASSWORD=admin123456 \
  minio/minio server /data --console-address ":9001"
```
