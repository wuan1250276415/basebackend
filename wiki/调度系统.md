# 调度系统

---

## 概述

BaseBackend 的调度系统位于 `basebackend-scheduler-parent/` 模块，集成 Camunda BPM 7.21.0 工作流引擎，支持业务流程定义、任务管理和历史查询。

---

## 模块结构

```
basebackend-scheduler-parent/
└── scheduler-camunda/       # Camunda BPM 集成
    ├── src/main/java/
    │   └── com/basebackend/scheduler/
    │       ├── config/      # Camunda 配置
    │       ├── delegate/    # 服务任务代理
    │       ├── listener/    # 任务监听器
    │       └── service/     # 调度服务
    └── src/main/resources/
        ├── application.yml
        ├── application-camunda.yml
        └── processes/       # BPMN 流程定义文件
```

---

## Camunda BPM 集成

### 配置

```yaml
# application-camunda.yml
camunda:
  bpm:
    database:
      schema-update: true
    admin-user:
      id: admin
      password: admin
    filter:
      create: All Tasks
    history-level: full          # 完整历史记录
```

### 核心概念

| 概念 | 说明 |
|------|------|
| 流程定义 (Process Definition) | BPMN 2.0 流程模型 |
| 流程实例 (Process Instance) | 运行中的流程 |
| 任务 (Task) | 需要人工处理的用户任务 |
| 服务任务 (Service Task) | 系统自动执行的任务 |
| 网关 (Gateway) | 流程分支（排他/并行/包含） |

## 流程定义管理

### 部署流程

```java
@Autowired
private RepositoryService repositoryService;

/**
 * 部署 BPMN 流程定义
 */
public void deployProcess(String name, InputStream bpmnStream) {
    repositoryService.createDeployment()
        .name(name)
        .addInputStream(name + ".bpmn", bpmnStream)
        .deploy();
}
```

### 查询流程定义

```java
/**
 * 查询所有已部署的流程定义
 */
public List<ProcessDefinition> listProcessDefinitions() {
    return repositoryService.createProcessDefinitionQuery()
        .active()
        .latestVersion()
        .orderByProcessDefinitionName().asc()
        .list();
}
```

## 任务管理

### 启动流程实例

```java
@Autowired
private RuntimeService runtimeService;

/**
 * 启动审批流程
 */
public String startProcess(String processKey, Map<String, Object> variables) {
    ProcessInstance instance = runtimeService.startProcessInstanceByKey(
        processKey, variables);
    return instance.getId();
}
```

### 查询待办任务

```java
@Autowired
private TaskService taskService;

/**
 * 查询当前用户的待办任务
 */
public List<Task> listMyTasks(String assignee) {
    return taskService.createTaskQuery()
        .taskAssignee(assignee)
        .active()
        .orderByTaskCreateTime().desc()
        .list();
}
```

### 完成任务

```java
/**
 * 完成任务（审批通过/拒绝）
 */
public void completeTask(String taskId, Map<String, Object> variables) {
    taskService.complete(taskId, variables);
}
```

## 服务任务代理

```java
/**
 * Camunda 服务任务代理
 * 在 BPMN 流程中被自动调用
 */
@Component("sendNotificationDelegate")
public class SendNotificationDelegate implements JavaDelegate {

    @Autowired
    private NotificationService notificationService;

    @Override
    public void execute(DelegateExecution execution) throws Exception {
        String userId = (String) execution.getVariable("userId");
        String message = (String) execution.getVariable("message");

        notificationService.send(userId, message);

        execution.setVariable("notificationSent", true);
    }
}
```

## 历史查询

```java
@Autowired
private HistoryService historyService;

/**
 * 查询已完成的流程实例
 */
public List<HistoricProcessInstance> listCompletedProcesses() {
    return historyService.createHistoricProcessInstanceQuery()
        .finished()
        .orderByProcessInstanceEndTime().desc()
        .list();
}

/**
 * 查询流程的活动历史（审批轨迹）
 */
public List<HistoricActivityInstance> getProcessHistory(String processInstanceId) {
    return historyService.createHistoricActivityInstanceQuery()
        .processInstanceId(processInstanceId)
        .orderByHistoricActivityInstanceStartTime().asc()
        .list();
}
```
