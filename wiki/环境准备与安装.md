## 基础设施安装

### 方式一：Docker Compose 一键安装（推荐）

项目提供了完整的 Docker Compose 编排文件，按功能分组在 `docker/compose/` 目录下。

```bash
# 1. 基础设施：MySQL + Redis
cd docker/compose/base
docker-compose up -d

# 2. 中间件：Nacos + RocketMQ + Seata + MinIO
cd ../middleware
docker-compose up -d

# 3. 可观测性（可选）：Prometheus + Loki + Tempo + Grafana
cd ../observability
docker-compose up -d
```

### 方式二：手动安装

#### MySQL 8

```bash
# macOS (Homebrew)
brew install mysql@8.0
brew services start mysql@8.0

# 创建数据库
mysql -u root -p
CREATE DATABASE basebackend DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### Redis 7

```bash
# macOS (Homebrew)
brew install redis
brew services start redis

# 验证
redis-cli ping
# 应返回 PONG
```

#### Nacos 2.x

```bash
# Docker 方式启动（推荐）
docker run -d --name nacos \
  -e MODE=standalone \
  -e NACOS_AUTH_ENABLE=true \
  -e NACOS_AUTH_TOKEN_EXPIRE_SECONDS=18000 \
  -p 8848:8848 \
  -p 9848:9848 \
  nacos/nacos-server:v2.3.0

# 验证：浏览器访问 http://localhost:8848/nacos
# 默认用户名/密码：nacos / nacos
```

#### RocketMQ 5.2.0

```bash
# 使用项目提供的 Docker Compose
cd docker/compose/middleware
docker-compose up -d rocketmq-namesrv rocketmq-broker rocketmq-dashboard
```

#### Seata 1.7.1

```bash
# 使用项目提供的配置
cd docker/seata-server
docker-compose up -d
```

#### MinIO

```bash
# Docker 方式启动
docker run -d --name minio \
  -p 9000:9000 \
  -p 9001:9001 \
  -e MINIO_ROOT_USER=admin \
  -e MINIO_ROOT_PASSWORD=admin123456 \
  minio/minio server /data --console-address ":9001"

# 管理控制台：http://localhost:9001
```

## IDE 配置

### IntelliJ IDEA（推荐）

#### 推荐插件

| 插件 | 用途 |
|------|------|
| Lombok | Lombok 注解支持 |
| MyBatisX | MyBatis Mapper XML 与接口跳转 |
| Spring Boot Assistant | Spring Boot 配置提示 |
| Maven Helper | Maven 依赖分析 |
| GitToolBox | Git 增强 |
| Alibaba Java Coding Guidelines | 阿里巴巴编码规范检查 |
| Docker | Docker 容器管理 |

#### IDEA 设置

1. **注解处理器**：`Settings > Build > Compiler > Annotation Processors > Enable annotation processing`
2. **编码设置**：`Settings > Editor > File Encodings > 全部设为 UTF-8`
3. **Maven 设置**：`Settings > Build > Build Tools > Maven > Maven home path` 指向本地 Maven
4. **JDK 设置**：`Project Structure > Project > SDK` 选择 JDK 17

## 环境变量配置

项目支持通过环境变量覆盖配置，以下是常用环境变量：

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `SPRING_PROFILES_ACTIVE` | `dev` | Spring 激活的配置文件 |
| `MYSQL_HOST` | `localhost` | MySQL 地址 |
| `MYSQL_PORT` | `3306` | MySQL 端口 |
| `MYSQL_DATABASE` | `basebackend` | 数据库名 |
| `MYSQL_USERNAME` | `root` | 数据库用户名 |
| `MYSQL_PASSWORD` | - | 数据库密码 |
| `REDIS_HOST` | `localhost` | Redis 地址 |
| `REDIS_PORT` | `6379` | Redis 端口 |
| `REDIS_PASSWORD` | - | Redis 密码 |
| `NACOS_SERVER_ADDR` | `localhost:8848` | Nacos 地址 |
| `ROCKETMQ_NAMESRV` | `localhost:9876` | RocketMQ NameServer |
| `SEATA_SERVER_ADDR` | `localhost:8091` | Seata Server 地址 |
| `MINIO_ENDPOINT` | `http://localhost:9000` | MinIO 端点 |
| `MAIL_HOST` | `smtp-mail.outlook.com` | SMTP 邮件服务器 |
| `MAIL_PORT` | `587` | SMTP 端口 |
| `LOGGING_LEVEL_ROOT` | `WARN` | 日志级别 |

> **安全提示**：生产环境中，敏感配置（密码、密钥等）应通过 Nacos 配置中心或密钥管理服务获取，避免明文写入配置文件。
